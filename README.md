# Carcharing — Система аренды автомобилей (каршеринг)
## Десктопное приложение для автоматизации аренды авто с расчётом стоимости, контролем лимитов и ролевой моделью «пользователь / администратор».

🚗 **Возможности приложения**
- Загрузка актуальных тарифов и наличия автомобилей из внешнего API
- Автоматический расчёт базовой стоимости аренды (цена за минуту/час)
- Спецусловия для длительных аренд (скидка от определённого времени)
- Контроль лимитов аренды на одного пользователя (макс. время, макс. пробег)
- Ролевая модель с разделением прав (пользователь / администратор)
- История поездок с сохранением в файл
- Администрирование: управление лимитами, тарифами, блокировка пользователей

---

## 📋 Функциональные требования

### 👤 Пользователь
- выбор доступного автомобиля по локации и классу;
- ввод пользователем даты, времени и длительности аренды;
- вывод требований к аренде (возраст, права, депозит);
- вывод истории поездок;
- сохранение истории в файл;
- автоматическая загрузка тарифов и наличия авто из внешнего API;
- автоматический расчёт итоговой стоимости (включая страховку и топливо).

---

## 👥 Ролевая модель

| Роль | Действия |
|------|----------|
| **Пользователь** | Просмотр доступных авто, выбор даты/времени, расчёт стоимости, просмотр истории, сохранение истории в файл |
